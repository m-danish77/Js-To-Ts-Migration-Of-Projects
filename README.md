# 🚀 Learning TypeScript: Backend Journey

This repository tracks my journey of transitioning from JavaScript to **TypeScript**, specifically focused on building robust and type-safe backend services. I've refactored three core backend projects from Node/Express to TypeScript to better understand interfaces, types, and modern development workflows.

---

## 🛠 Tech Stack

*   **Language:** TypeScript
*   **Runtime:** Node.js
*   **Framework:** Express.js
*   **Database:** MongoDB (via Mongoose)
*   **Tools:** TSC, Nodemon, Git

---

## 📂 Project Structure

This monorepo contains three distinct API projects, each demonstrating different aspects of TypeScript integration:

### 1. 📝 Typescript Todo List API
A foundational project refactored from a Node/Express base. It implements CRUD operations with strictly typed Mongoose models and Express controllers.
*   **Key Learnings:** Setting up `tsconfig.json`, defining Mongoose schemas with TypeScript interfaces, and basic error handling.

### 2. 💳 Typescript Expense Tracker API
A middle-tier API focused on managing financial transactions.
*   **Key Learnings:** Complex data types, working with environment variables in TS, and type-checking request bodies.

### 3. ✍️ Typescript Blog API
A more comprehensive API featuring post creation, user handling, and potentially relational data structures.
*   **Key Learnings:** Advanced interface inheritance, modular routing, and generic type usage for middleware.
